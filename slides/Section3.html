Novel Developments in SEIR(S) Models

My research expands on the traditional SEIR model in four principal ways.

  • Generalizing the types of spatial structure which may be applied to the data
  • Defining and implementing a general structure for specifying the exposure and recovery processes
  • Providing tools for model selection and diagnostic work
  • Building a (relatively) user friendly software implementation

Novel Developments in SEIR(S) Models

Spatial Generalization - Motivating Assumptions

  • The number of contacts between individuals in a spatial location follows a Poisson distribution.
  • When individuals travel, their contact behavior is well modeled by the contact behavior in their destination.
  • Contact between spatial locations can be expressed as a weighted sum of a number of "distance" metrics.

Spatial Generalization

Based on these assumptions and their associated parameters, we can derive a parametric form for the exposure probability at a particular time point and spatial location, which is required to construct the HM. The result is a flexible parameterization that can accommodate numerous population structures.

Intensity and Reinfection Processes

  • Generalization of intensity process structure has some history in the literature (i.e., Lekone and Finkenstadt, 2006), though it is generally presented differently for each problem. We have developed a general structure to tackle a wide variety of hypotheses, as well as to provide appropriate temporal bases for prediction in the absence of full information about disease behavior and control efforts.
  • The same structure is employed for reinfection, allowing the study of pathogen mutation and waning vaccine efficacy among other things.

Examples of the former are further explored in the case studies.

Model Selection and Diagnostic Tools

Model selection techniques are crucially important for observational data analysis, but can be difficult to implement for complicated hierarchical models.

  • Running many potential candidate models is impractical when each one requires hours or days.
  • Traditional methods like cross validation are not applicable.
  • These models tend to be implemented "by hand", and so lack general software in which to test multiple approaches.

We have developed a promising diagnostic tool that addresses these issues, though we must first define the "basic reproductive number".

Novel Developments in SEIR(S) Models

The Basic Reproductive Number

  • The basic reproductive number, or R0, is defined as "The expected number of secondary infections caused by a single infectious individual in an entirely susceptible population."
  • Epidemiologists and infectious disease experts use this quantity as a "threshold" parameters.
  • When R0 is below 1, the disease is not expected to spread.
  • When R0 is greater than one, a disease is expected to invade the population.

Novel Developments in SEIR(S) Models

The Basic Reproductive Number

  • This quantity, while incredibly useful, is somewhat removed from actual epidemic behavior.
  • R0 estimates depend entirely on the parametric form of the intensity process.
  • The measure is therefore extremely sensitive to model misspecification.
  • We propose an additional measure, R(EA), which considers the expected number of secondary infections caused by an infectious individual in a population under study.*

*This definition sounds similar to concepts like "effective R0" (Chowell et.al. 2004) and the replacement ratio (Hethcote 2000), but our development is mathematically distinct.

Software Development

  • While building these models can be quite difficult, it doesn't need to be.
  • All of this development has been compiled into an open source package for the R statistical computing environment.
  • The numerical heavy lifting is done in C++ with optional use of the OpenCL parallel processing framework.
  • While the user must still write some code, they interact with pre-made model types, much like standard statistical software.
  • This effort aims to expand both the scope of models which are practical to fit, and the group of people capable of easily using them.
